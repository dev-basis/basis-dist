# Basis Accounting System - Distribution Repository

This repository provides **official installer packages** for the Basis Accounting System.

## Editions

- **Desktop Edition (Free)**
  - Platform: Windows 64-bit
  - License: MIT License

- **Server Edition (Commercial)**
  - Platforms: Windows x64, Linux x64
  - License: Commercial license required

## Downloads

Installer files are available in the [Releases](../../releases) section of this repository.

## Notes

- Desktop Edition is free to use under the MIT License.
- Server Edition requires a valid commercial license for production use.
- Please refer to release notes for version changes and installation instructions.
